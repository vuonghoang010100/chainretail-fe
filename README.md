# Setup
  - Install **Nodejs** version: `20.11.1 LTS`.
  - Change directory to `chainretail-fe`.
  - Add *dependencies*: 
    ```shell
    npm install
    ```
  - Setup local env: create `.env.local` from `.env.local.example`
    ```shell
    cp .env.local.example .env.local
    ```
# Run
```shell
npm run dev
```

# Build
```shell
npm run build
```

# Lint
```shell
npm run lint
```
